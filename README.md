# 0xF Project

This project is built using [Next.js](https://nextjs.org) and [shadcn/ui](https://ui.shadcn.com). It is based on the installation guide from [shadcn/ui for Next.js](https://ui.shadcn.com/docs/installation/next).

## Getting Started

To start the development server, run:

```bash
pnpm dev
```

Visit [http://localhost:3000](http://localhost:3000) in your browser to view the application.
