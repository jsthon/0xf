# 0xF

A lightweight, open-source developer toolkit for practical development tasks.

[https://0xf.com](https://0xf.com)

## Getting Started

### Prerequisites

- Node.js >= 20
- pnpm (recommended via [Corepack](https://pnpm.io/installation#using-corepack))

### Install dependencies

```bash
pnpm install
```

### Start the development server

```bash
pnpm dev
```

Then open [http://localhost:3000](http://localhost:3000) in your browser.
